# Movies CRUD

Simple Node.js app using Express, Mongoose, and Pug.

## What it does
- Shows a list of movies
- Adds a new movie
- Updates a movie rating
- Deletes movies by rating

## Setup
1. Install packages:
   ```bash
   npm install
   ```
2. Add your MongoDB Atlas URI in `.env`:
   ```env
   MONGO_URI=your_mongodb_connection_string
   PORT=3000
   ```
3. Run the app:
   ```bash
   npm start
   ```
4. Open in browser:
   `http://localhost:3000`
